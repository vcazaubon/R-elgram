# Réelgram

Vault privé de Reels Instagram — sauvegardez, organisez et regardez vos Reels depuis n'importe quel appareil.

**Stack :** PWA Vite + React + TypeScript (frontend) · FastAPI + yt-dlp + ffmpeg (backend) · Supabase Cloud (auth + RLS) · vidéos stockées sur volume local · déploiement Docker via Coolify.
